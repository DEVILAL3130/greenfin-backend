This is the backend server for GreenFin, a loan management and investment platform.
The backend is built with Node.js, Express, and MongoDB, providing APIs for loan tracking, EMI payments, and loan marketplace functionality.

Features
.User authentication (JWT-based)
.Apply for loans
.Track active loans and EMI schedule
.Pay EMIs
.Download EMI receipts
.Loan marketplace for investors
.Alerts for upcoming EMIs
.Green impact tracking

Tech Stack
Node.js – Backend runtime
Express.js – Web framework
MongoDB – Database
jsonwebtoken – JWT authentication
dotenv – Environment variables management
nodemailer – Email notifications
node-cron – Scheduled tasks for loan updates
cloudinary- for the file or image uploads
